From Features to Explanations: A Study of Explainable and Generalisable AI-Generated Text Detection

An interpretable framework combining linguistic features, machine learning, and SHAP explanations for transparent AI text detection.

Abstract

As Large Language Models (LLMs) become increasingly embedded in everyday and academic contexts, reliably distinguishing AI-generated text from human writing has become an urgent challenge. Although many detection methods report high accuracy, most operate as black-box systems, providing limited insight into the linguistic evidence driving their predictions.

We present an explainable framework for AI-generated text detection that integrates linguistic feature engineering, machine learning, and Explainable AI (XAI). We evaluate 38 linguistic features and apply Recursive Feature Elimination (RFE) to derive dataset-specific optimal feature subsets for two benchmark corpora: the PAN-CLEF and COLING shared task datasets. Using an XGBoost classifier trained on these interpretable features, our approach achieves leaderboard-competitive performance on the PAN CLEF 2025 test set (F1 = 0.9748), comparable to top-performing systems while avoiding reliance on fine-tuned large language models.

However, performance degrades under cross-domain and cross-generator evaluation, particularly on COLING, highlighting persistent limitations in detector generalisability. To mitigate this, we investigate simple ensemble strategies that improve robustness on unseen domains. We further employ SHAP (SHapley Additive exPlanations) to provide global and instance-level explanations, identifying the linguistic features that drive predictions and clarifying why generalisation failures arise. Finally, we release an open-source Python package via PyPI that provides both predictions and interpretable explanations for individual texts, enabling transparent and informed use of AI-generated text detection in practice. Overall, our results show that explainable feature-based models can be highly competitive on shared-task benchmarks while providing critical transparency about their behaviour and limitations.
